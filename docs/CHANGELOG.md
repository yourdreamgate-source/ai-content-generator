# Changelog

All notable changes to AI Content Generator will be documented in this file.

## [1.0.0] - 2024-12-10

### Added
- Initial release
- User authentication (register, login, profile)
- Content generation with OpenAI GPT
- 8 pre-built templates:
  - Blog Post
  - Social Media Post
  - Email Copy
  - Product Description
  - Ad Copy
  - SEO Meta Description
  - YouTube Script
  - Newsletter
- Custom prompt generation
- Generation history with pagination
- Favorites system
- Copy and download functionality
- Credit-based usage system
- Admin dashboard with statistics
- User management (credits, roles, delete)
- Template management (CRUD operations)
- Responsive design
- Rate limiting
- JWT authentication
- SQLite database

### Security
- Password hashing with bcrypt
- JWT token authentication
- Rate limiting on API endpoints
- Input validation
- CORS configuration
