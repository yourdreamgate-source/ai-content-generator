# Screenshots Guide for CodeCanyon

CodeCanyon requires specific screenshots. Here's exactly what you need to capture:

## Required Screenshots (Minimum 3, Recommended 8)

### 1. THUMBNAIL (590x300 px) - REQUIRED
**File:** `thumbnail.png`
- Main preview image shown in search results
- Should show the dashboard or generate page
- Add your logo/branding
- Make it eye-catching with gradient background

### 2. PREVIEW IMAGE (590x300 px) - REQUIRED  
**File:** `preview.png`
- Larger version of thumbnail
- Can be same as thumbnail or more detailed

### 3. Screenshots (Each 590x300 or larger)

**Screenshot 1: Login Page**
- Show the beautiful gradient login screen
- URL: http://localhost:3000/login

**Screenshot 2: Dashboard**
- Show welcome message, stats, templates
- URL: http://localhost:3000/dashboard

**Screenshot 3: Generate Page (Empty)**
- Show the generate interface with template selected
- URL: http://localhost:3000/generate/1

**Screenshot 4: Generate Page (With Content)**
- Show generated content in the output area
- Generate some sample content first

**Screenshot 5: Templates Page**
- Show all 8 templates in grid view
- URL: http://localhost:3000/templates

**Screenshot 6: History Page**
- Show generation history with some entries
- URL: http://localhost:3000/history

**Screenshot 7: Admin Dashboard**
- Show admin stats and analytics
- URL: http://localhost:3000/admin

**Screenshot 8: Admin Users**
- Show user management table
- URL: http://localhost:3000/admin/users

## How to Take Screenshots

### Option 1: Browser Extension
1. Install "GoFullPage" Chrome extension
2. Navigate to each page
3. Click extension to capture full page
4. Crop to 590x300 or keep full size

### Option 2: Manual
1. Open browser DevTools (F12)
2. Click device toolbar icon
3. Set custom size: 1200x800
4. Take screenshot (Ctrl+Shift+P → "Capture screenshot")

### Option 3: Use Screely/Mockup Tools
1. Take regular screenshots
2. Upload to screely.com or similar
3. Add browser frame and shadow
4. Makes screenshots look more professional

## Screenshot Tips for Better Sales

1. **Use Real Content** - Generate actual content, don't show empty states
2. **Create Test Users** - Show user management with 5-10 users
3. **Show Credits** - Display credit balance prominently
4. **Highlight Features** - Use arrows/annotations if needed
5. **Consistent Branding** - Use same colors throughout
6. **High Quality** - Export as PNG, not JPG

## Creating the Thumbnail

Use Canva, Figma, or Photoshop:

1. Create 590x300 canvas
2. Add gradient background (indigo to purple)
3. Add mockup of dashboard (use screenshot)
4. Add text: "AI Content Generator"
5. Add bullet points: "GPT Powered • 8 Templates • Admin Panel"
6. Add "React + Node.js" badge
